# 트리 순회 방식

## 1. Inorder
> 순서
> - Left
> - Root
> - Right

## 2. Preorder
> 순서
> - Root
> - Left
> - Right
>

## 3. Postorder
> 순서
> - Left
> - Right
> - Root
>
